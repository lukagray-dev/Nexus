01

Create Firebase Project

Set up a free Firebase project at console.firebase.google.com. Enable Realtime Database and configure authentication. Download your configuration file.

02

Install Agent on Child Device

Download and install the Nexus Child app on the target device. Grant all required permissions including accessibility, location, and device admin.

03

Configure Connection

Enter your Firebase credentials in the Agent app. The device will automatically register and begin syncing data to your Firebase database.

04

Install Dashboard

Download Nexus Parent on your monitoring device (Windows, macOS, or Linux). Connect using the same Firebase credentials to start monitoring.

05

Start Monitoring

You're all set! Access real-time data, view activity logs, and control the device remotely from your Dashboard. Enable stealth mode for invisible operation.

Architecture

How Nexus Works

A sophisticated two-part system designed for seamless operation

01

Nexus Child (Agent)

The Agent is installed on the child's device. Built with native technologies (Kotlin for Android, Swift for iOS), it operates at the system level with minimal resource usage. Features include:

  • Background service operation
  • System-level integration
  • Stealth mode capability
  • Auto-recovery from termination
  • Survives device reboots
02

Nexus Parent (Dashboard)

The Dashboard is your command center, available on desktop (Windows, macOS, Linux) and mobile (Android, iOS). Built with Electron for desktop and native frameworks for mobile. Features include:

  • Real-time data visualization
  • Multi-device management
  • Command dispatch system
  • Historical data analysis
  • Secure WebRTC connections
03

Secure Communication

Data flows securely between Agent and Dashboard using Firebase Realtime Database for signaling and WebRTC for peer-to-peer media streaming. The architecture ensures:

  • End-to-end encrypted connections
  • Low-latency media streaming
  • Reliable command delivery
  • Automatic reconnection
  • Minimal bandwidth usage
